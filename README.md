# portfolio-saint-mathieu-makon

##  Description
Ce projet est un **portfolio full-stack développé avec Next.js** dans le cadre d’un projet académique.  
Il permet de présenter mes compétences, mes projets et d’interagir avec les utilisateurs via un système d’authentification et de témoignages.

Le projet intègre :
- Un frontend moderne avec React / Next.js
- Un backend avec Next.js API Routes
- Une gestion d’état globale avec Redux Toolkit
- Une communication API avec Axios
